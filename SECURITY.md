# Security Policy

## Supported Versions

Security fixes are provided for the latest release line.

## Reporting a Vulnerability

Please report vulnerabilities privately by opening a security advisory in GitHub:

- Go to the repository Security tab.
- Select "Advisories" and click "Report a vulnerability".
- Include reproduction steps, impact, and affected versions.

If GitHub advisories are unavailable, contact: silentglasses@proton.me

## Response Targets

- Initial acknowledgment: within 72 hours
- Triage decision: within 7 days
- Status updates: at least every 7 days until resolved

## Disclosure

Please do not open public issues for unpatched vulnerabilities.
Public disclosure is requested only after a fix is released.
